{% blocktrans %} The langi web application was created for all users who wish to develop their language skills in a pleasant and convenient way. The application allows you to view and learn the cards you have created using one of the most popular SRS (Spaced Repetition System) algorithms, SM-2. In addition, apart from the classic approach that allows you to add new flashcards, the user also has the option of automatically creating word sets from the imported files. The application supports both text document files as well as image files such as .png or .jpg. The presence of mini-games, with the help of which the user can learn new words or expressions, also contributes to pleasant learning. For the convenience of the user, the application supports 16 interface languages ​​and two color modes. The "light" mode, which is the default mode, and the "dark" mode, which, due to a number of tests, will certainly be better suited for night use of the application. {% endblocktrans %}


{% blocktrans %} The application was created with the use of Django technology and Django REST framework, as well as with the help of the OpenCV (Open Source Computer Vision) library. {% endblocktrans %}
